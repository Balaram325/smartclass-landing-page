# 🌐 SmartClass Landing Page

A modern and responsive landing page built for the **SmartClass AI** attendance system.

Designed using **Flask**, **HTML**, and **CSS** with a clean UI showcasing the platform's AI-powered features.

---

## 🚀 Live Website

https://smartclass-landing-page.onrender.com

---

## ✨ Features

- Modern Hero Section
- Responsive Navigation
- AI Feature Showcase
- Technology Stack Section
- Teacher Workflow
- Student Workflow
- Mobile Responsive Design
- Clean Footer
- Smooth UI

## 🛠 Built With

- Flask

- Responsive Design
